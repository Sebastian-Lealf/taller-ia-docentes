Antes de empezar: que son estas herramientas y que es un prompt

Esta introduccion sirve para nivelar al grupo. La idea central es simple: una herramienta de IA no es "mejor" en abstracto. Es mejor o peor segun la tarea, el tipo de archivo, la necesidad de fuentes, la privacidad, el presupuesto y el nivel de control que requiere el investigador.

Frase para abrir: "No vamos a escoger una IA ganadora. Vamos a escoger la herramienta correcta para cada momento del trabajo academico: explorar, leer, analizar, escribir, visualizar, citar y presentar".

Que es una herramienta de IA aplicada

Es una aplicacion que usa modelos de inteligencia artificial para ayudar en una tarea concreta: conversar, resumir, programar, buscar evidencia, crear imagenes, generar diapositivas, analizar datos o transformar texto en diagramas. En investigacion, su valor no esta en "hacer el trabajo por nosotros", sino en acelerar borradores, organizar informacion y abrir caminos que luego el experto verifica.

Que es un prompt

Un prompt es una instruccion escrita que le dice a la IA que debe hacer, con que contexto, bajo que criterios y en que formato debe responder. Un buen prompt reduce ambiguedad; no garantiza verdad. Por eso debe combinarse con verificacion experta.

Prompt debil

"Hazme un resumen"

Es ambiguo: no dice para quien, de que extension, con que enfoque, si debe citar, si debe criticar o si debe conservar terminos tecnicos.

Prompt fuerte

Rol + contexto + tarea + formato

Define el papel de la IA, el publico, los criterios de calidad, los limites y la forma exacta de salida.

Partes de un buen prompt

PartePregunta que respondeEjemplo
RolDesde que perspectiva debe responder?Actua como metodologo de investigacion educativa.
ContextoQue informacion necesita conocer?Curso de maestria, docentes investigadores, Colombia.
TareaQue debe producir?Formula pregunta, objetivos, variables y metodo.
CriteriosQue cuenta como buena respuesta?No inventes citas; separa evidencia de inferencia.
FormatoComo debe entregar?Tabla con columnas: autor, metodo, muestra, hallazgos.

IteracionComo se mejora la respuesta?Ahora reduce a 200 palabras y agrega limitaciones.

Mapa de herramientas

Selecciona segun el problema, no segun la moda. Esta lista esta pensada para docentes e investigadores de alto nivel que necesitan trazabilidad, fuentes y control.

Analisis y escritura

ChatGPT

Analiza archivos, crea tablas y graficas, ayuda a redactar, depurar argumentos y generar codigo para Python o Colab.

Archivos largos

Claude

Bueno para leer documentos extensos, comparar versiones, crear matrices de evidencia y redactar con instrucciones de estilo.

Google Workspace

Gemini

Util para investigacion multimodal, imagenes, integracion con ecosistema Google y apoyo en documentos, hojas y correos.

Fuentes propias

NotebookLM

Trabaja con PDFs, Docs, Slides, URLs, YouTube y audio. Ideal para construir un cuaderno basado en fuentes verificables.

Presentaciones

Gamma

Genera presentaciones, documentos y paginas a partir de prompts, esquemas, archivos o URLs.

Busqueda con citas

Perplexity

Respuestas con fuentes, espacios de investigacion, busqueda web y academica, archivos y conectores en planes empresariales.

Revision sistematica

Elicit

Busca literatura, filtra articulos, extrae datos y ayuda a construir revisiones sistematicas o estados del arte.

Evidencia cientifica

Consensus

Motor de busqueda academica con respuestas basadas en papers, grafo de citas, biblioteca y analisis de articulos.

Lectura de papers

SciSpace

Explica papers, conversa con PDFs, ayuda a entender metodos, ecuaciones y conceptos densos.

Bibliografia

Zotero

Gestor bibliografico. No es IA generativa por si mismo, pero es clave para ordenar evidencia y exportar BibTeX/RIS.

Diseno visual

Canva

Materiales didacticos, infografias, piezas para clase, plantillas y funciones de IA para diseno.

Diagramas

Napkin AI

Convierte texto en esquemas visuales, diagramas y mapas conceptuales para explicar ideas complejas.

Flujo completo: de una idea a paper, graficas y presentacion

Este es el hilo narrativo mas potente para la conferencia: no mostrar herramientas aisladas, sino una cadena de trabajo academico. Puedes decirles: "vamos a producir un miniarticulo verificable, con datos, graficas, bibliografia, LaTeX y diapositivas".

Paso 1
GPT con internet o Perplexity: explorar el tema, localizar datos recientes, autores, cifras y debates. Resultado: pregunta inicial y mapa del problema.
Paso 2
Elicit / Consensus / SciSpace: buscar papers, extraer evidencia y construir estado del arte. Resultado: matriz de literatura.
Paso 3
Zotero: guardar referencias, revisar metadatos, generar BibTeX o RIS. Resultado: biblioteca trazable.
Paso 4
ChatGPT / Claude: generar o limpiar dataset demostrativo, definir variables y limites. Resultado: CSV listo.
Paso 5
Google Colab: analizar datos y producir graficas reproducibles. Resultado: PNGs, hallazgos y codigo.
Paso 6
Claude / ChatGPT: integrar literatura, metodo, resultados y discusion. Resultado: borrador academico.
Paso 7
ChatGPT / Claude: exportar a LaTeX con secciones, tablas, figuras y referencias BibTeX. Resultado: main.tex + references.bib.
Paso 8
Gamma / Canva / Napkin: convertir el paper en presentacion, infografias y diagramas para clase. Resultado: material de socializacion.

1. Titulo
2. Resumen de 180-220 palabras
3. Palabras clave
4. Introduccion
5. Estado del arte sintetico
6. Metodo
7. Resultados descriptivos
8. Discusion
9. Limitaciones
10. Conclusiones

Reglas:
- No inventes citas ni resultados.
- Donde falte una cita, escribe [CITA PENDIENTE].
- Donde vaya una figura, escribe [FIGURA 1: descripcion].
- Mantén tono academico sobrio.
- Aclara que el dataset es sintetico si fue generado para el taller.
- No exageres causalidad: usa "se observa", "sugiere", "en este ejercicio descriptivo".

Evitar: caricaturas, estudiantes sonrientes genericos, laboratorios futuristas, exceso de texto, manos deformes, logos, marcas de agua visibles.

Prompt para editar una imagen

Edicion de imagen
Usa la imagen cargada como base y transformala en material didactico.

Mantén la estructura principal, pero:
1. Limpia el fondo y mejora contraste.
2. Convierte el estilo a ilustracion editorial universitaria.
3. Agrega zonas de enfoque con colores suaves.
4. Deja espacio superior para titulo.
5. No cambies el significado cientifico de la imagen.
6. No agregues texto pequeno ilegible.

Resultado esperado: una imagen clara para una diapositiva de clase, no una pieza publicitaria.

Checklist etico para cerrar

Este cierre ayuda a que la conferencia no parezca una feria de herramientas. La IA entra al trabajo academico solo si mejora claridad, trazabilidad y calidad.

  • Privacidad: no subir datos sensibles, historias clinicas, evaluaciones nominales ni entrevistas identificables sin autorizacion.
  • Autoría: declarar el uso de IA cuando la revista, universidad o convocatoria lo exija.
  • Verificacion: toda cita, DOI, dato estadistico o afirmacion fuerte debe revisarse en la fuente original.
  • Sesgo: pedir a la IA que busque contraargumentos, no solo evidencia favorable.
  • Reproducibilidad: guardar prompts, versiones, criterios de busqueda y fecha de consulta.
  • Pedagogia: evaluar el razonamiento del estudiante, no solo el producto final pulido por IA.
